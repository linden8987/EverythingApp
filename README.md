# EverythingApp
An app that literally contains everything you would imagine. That's right! it has a built-in web browser, file explorer, settings, emulators built into it (yes if you want to run Windows XP on your PC, you can run it this way), and more.
## Web Browser
The web browser is based on Chromium except it's portable (literally, you can run a web browser in the recovery environment).
There is downloads built-in, tabs, extensions and basically everything you can imagine for a regular browser.
## File Explorer
Basically functions like normal explorer, but usually you have to right click on the files to open them with the associated program, cause it might just try to open it inside of the app, which does not work.
## Emulator Built-in
Based off QEMU, this will literally allow you to run any operating system. You can change the processor, give it more RAM, change the architecture, and everything you could imagine. There's even some stuff that QEMU can't do that this will.
## Works On WinRE
That's right! Your recovery environment will be really, really really really really really upgraded.
